Student_management_app

A Java-based web application developed to efficiently manage student records with secure authentication and complete CRUD (Create, Read, Update, Delete) functionality. The application simplifies student data management through an intuitive interface and a MySQL backend.

---

## 📌 Overview

The Student Management System is designed to help administrators manage student information digitally. It allows users to add, update, view, search, and delete student records while ensuring data consistency through MySQL database integration.

---

## 🚀 Features

- 🔐 Admin Login Authentication
- ➕ Add New Student
- ✏️ Update Student Details
- ❌ Delete Student Records
- 🔍 Search Student Information
- 📋 View All Students
- 💾 MySQL Database Integration
- 🎨 Responsive User Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/nikhitha164/Student-Management--app.git
```

### Import the Project

- Open Eclipse IDE
- Import Existing Dynamic Web Project
- Configure Apache Tomcat
- Configure MySQL Database
- Update JDBC credentials
- Run the project on Tomcat Server
